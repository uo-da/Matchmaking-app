# 機能一覧

## 概要

エンジニア向けマッチングアプリのMVPで提供する主要機能を、利用者の操作順に沿って整理しています。

## 一覧

| 分類 | 機能 | 内容 |
| --- | --- | --- |
| 認証 | GitHub連携ログイン | GitHubユーザ名を入力してログインします。 |
| 認証 | セッション復元 | ブラウザの保存情報から前回のログイン状態を復元します。 |
| 認証 | ログアウト | 現在のセッションを解除します。 |
| 年齢確認 | 18歳以上の確認 | 利用開始前に年齢を確認します。 |
| プロフィール | プロフィール登録 | 表示名や自己紹介などを登録します。 |
| プロフィール | プロフィール編集 | 登録済みプロフィールを更新します。 |
| プロフィール | 年齢・自己紹介の設定 | 年齢と自己紹介文を管理します。 |
| プロフィール | 技術スタック・趣味の設定 | 技術タグと趣味情報を登録します。 |
| プロフィール | GitHubの草表示 | プロフィール画面でGitHubの活動イメージを表示します。 |
| ユーザ探索 | ユーザ一覧表示 | マッチ候補のユーザを一覧で表示します。 |
| ユーザ探索 | 技術タグ検索 | 指定した技術タグを含むユーザを絞り込みます。 |
| ユーザ探索 | 経験年数フィルタ | 経験年数の条件で候補を絞り込みます。 |
| ユーザ探索 | おすすめ候補の表示 | 自分の技術スタックと相性の良い候補を見せます。 |
| マッチング | いいね送信 | 気になるユーザへ通常のいいねを送ります。 |
| マッチング | スーパーライク送信 | 強い関心を示すスーパーライクを送ります。 |
| マッチング | 相互いいねでマッチ成立 | 双方の意思が一致したらマッチ状態にします。 |
| マッチング | マッチ一覧表示 | 成立済みのマッチ相手を一覧表示します。 |
| チャット | メッセージ送受信 | マッチ済みユーザとリアルタイムに会話します。 |
| チャット | 通知同期 | BroadcastChannel または localStorage を使って更新を同期します。 |
| チャット | 未読や更新の反映 | 新着メッセージや変更を画面へ反映します。 |

## 補足

- 画面遷移の起点はログイン画面です。
- 年齢確認を完了しないとメイン機能には進めません。
- マッチ成立後にチャット機能が利用可能になります。
